# 记忆系统使用说明

## 快速开始

### 安装和配置

确保项目已正确配置：

```python
from cognix.utils.config import config

# config.home_path 是记忆文件存储位置
# config.db_path 是SQLite索引文件位置
```

### 基本使用

```python
from cognix.core.memory_system import get_memory_system

memory = get_memory_system()

# 添加记忆
memory.add_memory("用户信息", "姓名：张三，28岁")

# 添加持久记忆
memory.add_persistent_memory("常用命令", "git status\ngit commit")

# 搜索
results = memory.search_memory("张三")

# 获取上下文
context = memory.get_daily_context(days_back=3)
```

## 使用Agent接口

```python
from cognix.interfaces.agent import AgentMemoryClient

client = AgentMemoryClient()

# 记住内容
client.remember("项目笔记", "正在开发记忆系统")

# 持久化记忆
client.remember("API文档", "GET /api/users", persistent=True)

# 搜索
result = client.recall("API")

# 获取上下文（包含短期记忆）
session_id = "my-session-123"
context = client.contextualize(session_id)
```

## 短期记忆使用

```python
# 添加
client.note_short_term(session_id, "task", "完成测试")
client.note_short_term(session_id, "current_step", "步骤2")

# 获取
short_term = client.get_short_term(session_id)

# 清除
client.clear_session(session_id)
```

## 直接编辑Markdown文件

记忆系统的一大优势是您可以直接编辑Markdown文件！

### 编辑每日记忆

找到 `memory/YYYY-MM-DD.md` 文件，直接用文本编辑器编辑：

```markdown
# 2026-04-25 日志

## 我的笔记

直接在这里写笔记，记忆系统会自动索引！

## 更多内容

- 项目计划
- 重要提醒
```

### 编辑持久记忆

编辑 `MEMORY.md` 文件：

```markdown
# 持久记忆

## 我的信息

姓名：张三
邮箱：zhangsan@example.com

## 常用链接

- GitHub: https://github.com
- 文档: https://docs.example.com
```

### 重建索引（如果需要）

编辑完文件后，重建索引确保最新：

```python
memory.rebuild_index()
# 或
client.rebuild_index()
```

## 完整示例

### 聊天机器人记忆

```python
from cognix.interfaces.agent import AgentMemoryClient

def chat():
    client = AgentMemoryClient()
    session_id = f"chat-{int(time.time())}"
    
    print("你好！我是记忆助手")
    
    while True:
        user_input = input("> ")
        
        if user_input.lower() == 'quit':
            break
        
        # 记录对话
        client.note_short_term(session_id, "last_input", user_input)
        
        # 搜索相关记忆
        memories = client.recall(user_input)
        
        # 获取上下文
        context = client.contextualize(session_id)
        
        # 简单的回复逻辑
        if memories:
            print(f"我记得相关内容：{memories[0]['text'][:100]}")
        else:
            print(f"收到！我记住了这条信息。")
            client.remember(f"对话-{len(context)}", user_input)

chat()
```

### 记录项目进展

```python
from datetime import datetime
from cognix.core.memory_system import get_memory_system

memory = get_memory_system()

def log_progress(task, details):
    today = datetime.now().date()
    heading = f"项目进展 - {task}"
    content = f"日期：{today}\n详情：{details}"
    memory.add_memory(heading, content)

# 使用
log_progress("记忆系统", "完成了Markdown存储重构")
log_progress("单元测试", "添加了搜索功能的测试")
```

## 常见问题

### Q: 索引损坏了怎么办？

A: 重建索引即可：

```python
memory.rebuild_index()
```

### Q: 如何备份我的记忆？

A: 直接备份 `memory/` 目录和 `MEMORY.md` 文件即可。SQLite索引不需要备份，可以重建。

### Q: 记忆文件在磁盘上什么位置？

A: 由 `config.home_path` 决定，通常在用户主目录下的 `.cognix/` 目录。

### Q: 可以在多个设备间同步记忆吗？

A: 可以！Markdown文件非常适合用Git、Dropbox等工具同步。

### Q: 短期记忆会持久化吗？

A: 不会。短期记忆只在内存中，会话结束就会消失。需要持久化请用 `add_persistent_memory()` 或 `remember(..., persistent=True)`。
